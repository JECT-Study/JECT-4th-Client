# 📊 Impact Influencer Platform
### 임팩트 있는 인플루언서를 찾고, 그리고 되다

---

## 🚀 About

글로벌 인플루언서 마케팅 시장은 2025년 약 **325억 달러 규모**로 성장 중이며,  
국내 MZ세대의 **72.3%는 인플루언서 추천을 통해 실제 구매 경험**을 가지고 있습니다.

하지만 여전히 시장에는 큰 문제가 존재합니다.

- 마케터는 인플루언서 검증에 **1시간 이상을 수동으로 소비**
- 마케팅 예산의 약 **14%가 가짜 데이터(허위 구독자 등)** 로 낭비
- 브랜드는 “진짜 영향력”을 가진 크리에이터를 찾기 어려움
- 크리에이터는 데이터 기반 성장 전략 없이 운영

또한 한국 스마트폰 사용자의 94%가 동영상 앱을 사용하며,  
가장 많은 이용자를 보유한 플랫폼은 유튜브입니다.

👉 우리는 이 지점에서 문제를 정의했습니다.

> “브랜드는 임팩트 있는 인플루언서를 찾기 어렵고  
> 인플루언서는 자신의 영향력을 증명하기 어렵다.”

---

## 🎯 Our Mission

**Impact Influencer Platform**은  
👉 *“임팩트 있는 인플루언서를 찾기 + 되기”*  
이 두 가지 문제를 동시에 해결하는

📌 **유튜브 중심 데이터 기반 인플루언서 인텔리전스 플랫폼**입니다.

| 🏢 브랜드 / 마케터 | 신뢰 가능한 인플루언서 빠르게 탐색 |

## 🔑 Core Features

### 1. 데이터 기반 인플루언서 탐색
- 카테고리 / 구독자 / 참여율 기반 검색
- 가짜 구독자 탐지
- 신뢰도 점수 제공

➡️ “광고 가능한 채널”이 아닌  
➡️ “임팩트 있는 채널”을 발견

---

### 2. 채널 성장 인사이트
- 조회수 / 참여율 시계열 분석
- Shorts vs Long-form 성과 비교
- 시청 유지율 분석
- CTR 및 오디언스 분석

➡️ 감이 아닌 데이터로 채널 운영

---

### 3. Fraud Detection (신뢰도 분석)

플랫폼은 다음과 같은 패턴을 분석합니다.

- 비정상적인 구독자 급증
- 참여율 대비 조회수 불균형
- 성장 곡선 이상 탐지

이를 기반으로

📊 **신뢰도 점수 (0~100)** 제공

➡️ 광고 리스크 최소화

---

### 4. 인플루언서 Impact Score

단순 조회수가 아닌

- 참여율
- 성장률
- 시청 지속도
- 협업 효과

등을 종합한

📈 **실질 영향력 지표 제공**

---

### 5. 경쟁사 협업 인플루언서 분석

브랜드는 다음을 확인할 수 있습니다.

- 경쟁사가 협업한 크리에이터
- 협업 콘텐츠 트렌드
- 콘텐츠 카테고리 흐름

➡️ 캠페인 전략 수립 가능

---

### 6. 협업 커뮤니케이션

플랫폼 내에서

- 협업 제안
- 수락 / 거절 / 역제안
- 협업 진행 관리

까지 가능

➡️ 이메일 / DM 의존 제거

---

### 7. AI 기반 캠페인 & 콘텐츠 지원 *(Post-MVP)*

- 캠페인 기획 지원
- 콘텐츠 방향 제안
- 성장 전략 가이드
